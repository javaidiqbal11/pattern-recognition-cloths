## Pattern Recognition 

### Setup
use Python 3.10.10

Install packages
```shell
pip install -r requirements.txt
```

### CNN
Train cnn model
```shell
python cnn.py
```
### Kaggle Notebooks
There are two notebooks available to run the code and check output.
1. fabric-cnn.ipynb 
2. fabric-effiecientnet.ipynb


### ResNet
Train cnn model
```shell
python resnet.py
```

### GCNN
Check final outcomes 
```shell
python gcnn.py
```
